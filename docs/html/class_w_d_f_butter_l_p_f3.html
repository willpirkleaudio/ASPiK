Detailed Description

The WDFButterLPF3 object implements a 3rd order Butterworth ladder filter. NOTE: designed with Elsie www.TonneSoftware.comm.

Audio I/O:

  • Processes mono input to mono output.

Control I/F:

  • none - this object is hard-wired.
