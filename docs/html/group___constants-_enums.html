Equalization.

Compressor, expander, limiter, etc.

Pitch processing.

aaxPlugInCategory_Reverb 

Reverberation and room/space simulation.

aaxPlugInCategory_Delay 

Delay and echo.

aaxPlugInCategory_Modulation 

Phasing, flanging, chorus, etc.

aaxPlugInCategory_Harmonic

Noise reduction.

aaxPlugInCategory_Dither 

Dither, noise shaping, etc.

aaxPlugInCategory_SoundField 

Pan, auto-pan, upmix and downmix, and surround handling.

aaxPlugInCategory_HWGenerators 

Fixed hardware audio sources such as SampleCell.

aaxPlugInCategory_SWGenerators 

Virtual instruments, metronomes, and other software audio sources.

Plug-ins wrapped by a thrid party wrapper except synth plug-ins which = AAX_PlugInCategory_SWGenerators.

Special effects.

◆ stateReg

enum stateReg

Use this enum to easily access z^-1 state values inside of arrays. For some structures, fewer storage units are needed. They are divided as follows:

  • Direct Forms: we will allow max of 2 for X (feedforward) and 2 for Y (feedback) data
  • Transpose Forms: we will use ONLY the x_z1 and x_z2 registers for the 2 required delays
  • enum stateReg { x_z1, x_z2, y_z1, y_z2, numStates };
