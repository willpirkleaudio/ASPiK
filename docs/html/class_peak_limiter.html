Detailed Description

The PeakLimiter object implements a simple peak limiter; it is really a simplified and hard-wired versio of the DynamicsProcessor.

Audio I/O:

  • Processes mono input to mono output.

Control I/F:

  • setThreshold_dB(double _threshold_dB) to adjust the limiter threshold
  • setMakeUpGain_dB(double _makeUpGain_dB) to adjust the makeup gain
